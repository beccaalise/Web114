Chapter 3: JavaScript Questions and Exercises

Questions

Q1: When would you use an "if" statement?

A: You would use an "if" statement when you want to execute different code depending on a condition. For example, checking if a user is logged in before showing a dashboard.

Q2: When comparing values, in what data type is the result stored?

A: The result is stored as a boolean.
